# Boilerplate Python

Espelho do diretório `boilerplate-laravel`: **Queries** + **Services** (web e API) para o recurso exemplo **Carro**, com SQLAlchemy 2 e o mesmo formato de retorno (`sucesso`, `dados`, `erros`).

Instalação local:

```bash
cd boilerplate-python
pip install -e .
```

Configure a URL do banco em `DATABASE_URL` (por exemplo `sqlite:///./carros.db` ou um DSN Postgres/MySQL).
